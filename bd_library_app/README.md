# bd_library_app

Application Flutter **Bibliothèque BD** du projet [MyBookShelves](../README.md).

Voir le [README principal à la racine du dépôt](../README.md) pour la description du projet, les fonctionnalités, les prérequis et les instructions de démarrage et de build.

## Commandes rapides

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run
```
